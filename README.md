# api-go

## Description

CRUD API rest with Golang, Postgresql, Authentication, Docker and Tests

## Getting Started

### Dependencies

* docker-compose version '3.9'

### Installing

* clone the repository
* set the environment variables

### Executing program

* docker-compose build
* docker-compose up
